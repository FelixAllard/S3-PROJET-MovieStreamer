# This is a guide on how to run things for development
## First, run the docker compose
```text
docker compose up -d
```
## Next, just run 
Go in intellij, at the top right, launch the message.main using intelliJ
## APP IS UP!
If you want to test the endpoints, use the following link in your browser
```text
http://localhost:8888/q/swagger-ui
```
